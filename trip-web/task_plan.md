# Trip Web 应用开发计划

## 项目概述
- **项目名称**: Trip Planner Web App
- **目标**: 创建一款旅行计划应用，支持行程规划、实时导航、美食推荐等功能
- **技术栈**: React + Vite + TailwindCSS + Leaflet (地图)
